# Premium Car Dealership Website - Build Plan

## Overview
Build a premium luxury car dealership website inspired by Big Motoring World but with a unique, modern Tesla/BMW/Mercedes-Benz aesthetic. Frontend-focused with mock data integration points for backend services.

## Reference
- https://www.bigmotoringworld.co.uk

## Tech Stack
- Next.js 15 + React + TypeScript + Tailwind CSS
- Framer Motion + GSAP for animations
- shadcn/ui components
- Mock data for all features (backend-ready architecture)

## Skill: vibecoding-webapp-swarm

---

## Stage 1 - Reference Analysis & Asset Collection
- Visit reference website to understand layout patterns
- Search for luxury car images for the dealership inventory
- Collect brand assets and inspiration

## Stage 2 - Design Phase (vibecoding-webapp-swarm Design Phase)
- Create design.md with complete design system
- Define all sections, animations, interactions
- Color palette: Black/White/Dark Gray/Metallic Silver/Luxury Blue

## Stage 3 - Website Building (vibecoding-webapp-swarm Build Phase)
- Initialize Next.js project with shadcn/ui
- Install animation libraries (Framer Motion, GSAP)
- Build all pages and components:
  - Homepage (Hero, Search, Featured Cars, Why Choose Us, Reviews, Finance, Blog, Contact)
  - Inventory Page (Grid/List view, Filters, Sorting)
  - Vehicle Detail Page (Gallery, Specs, Finance Calculator, Enquiry)
  - Sell Your Car Page (Multi-step form)
  - Finance Page (Calculator, FAQ)
  - User Dashboard (Saved cars, comparisons, enquiries)
- Implement all animations, glassmorphism, 3D effects
- Responsive design for all breakpoints

## Stage 4 - Build & Deploy
- Build and deploy to production
